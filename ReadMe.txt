This contains the set of instructions required to build and run addressBook application.

Building the Project.
====================

Pre-requisites
--------------
JDK 1.6.x or higher
Maven 2 or higher
java_home and m2_home has been set.

Steps:
------
1 - download and unzip the project.
2 - Go to the project home directory of addressbook, and run 
    $ mvn clean package

Running the application.
========================

on the builld window type 
	mvn exec:java
		
Please follow the UI prompts
